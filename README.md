# 🦉 Aulas de programação competitiva - OwlBI

Bem-vindos ao repositório oficial das aulas de preparação para a **Olimpíada Brasileira de Informática (OBI)**!

## 🎯 Objetivo
Fornecer a base teórica e prática em algoritmos e estruturas de dados para que você possa competir em alto nível na OBI e em outras competições de programação.

## 📅 Cronograma de aulas
Acesse diretamente o material, os slides e os exercícios de cada aula pelos links abaixo:

| Aula | Tópicos abordados | Acesso rápido |
| :---: | :--- | :---: |
| **01** | Programação competitiva, I/O e variáveis em C++ | [Acessar 📂](./conteudos/aula01/) |
| **02** | Estruturas condicionais (if/else), loops (for/while) e funções | [Acessar 📂](./conteudos/aula02/) |
| **03** | Arrays e vetores | Em breve |

## 📂 Estrutura do repositório
Todo o material está dividido por tópicos na pasta `conteudos/`. Dentro do diretório de cada aula, você encontrará:
* **PDFs** dos slides utilizados nas apresentações.
* **READMEs** com o resumo dos conceitos e links diretos para os problemas recomendados com seus respectivos níveis de dificuldade.
* **Códigos** de exemplo e gabaritos de resolução.

## 💻 Plataformas utilizadas
Durante nossos treinos, resolveremos problemas principalmente nestas plataformas:
* [Neps Academy](https://neps.academy/)
* [Beecrowd](https://www.beecrowd.com.br/)
* [Codeforces](https://codeforces.com/)

## 🤝 Como contribuir
Sinta-se à vontade para fazer um *fork* deste repositório e abrir um *pull request* adicionando suas soluções para os desafios propostos em aula na pasta `solucoes_alunos/`. Toda contribuição e discussão de lógica é muito bem-vinda!
