# 🦉 Aulas de programação competitiva - OwlBI

Bem-vindos ao repositório oficial das aulas de preparação para a **Olimpíada Brasileira de Informática (OBI)**!

## 🎯 Nosso objetivo
Fornecer a base teórica e prática em algoritmos e estruturas de dados para que você possa competir na OBI e em outras competições de programação.

## 📂 Como navegar neste repositório
Todo o material está dividido por tópicos na pasta `conteudos/`. Dentro da pasta de cada aula, você encontrará:
* PDFs dos slides utilizados nas aulas.
* READMEs com o resumo dos conceitos e links para os problemas recomendados.
* Códigos de exemplo e templates.

## 💻 Plataformas utilizadas
Durante nossos treinos, resolveremos problemas principalmente nestas plataformas:
* [Neps Academy](https://neps.academy/)
* [Beecrowd](https://www.beecrowd.com.br/)
* [Codeforces](https://codeforces.com/)

## 🤝 Como contribuir
Sinta-se à vontade para fazer um *fork* deste repositório e abrir um *pull request* adicionando suas soluções para os desafios propostos em aula na pasta `solucoes_alunos/`.
