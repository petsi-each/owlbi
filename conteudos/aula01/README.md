# Aula 01: Programação competitiva e introdução a C++

## 🦉 Sobre a aula
Bem-vindos ao primeiro encontro da OwlBI! Nesta aula, apresentamos o mundo da programação competitiva, compartilhamos nossas experiências e demos os primeiros passos na linguagem C++.

## 🎯 Tópicos abordados
1. **O que é programação competitiva:** OBI, maratonas e os benefícios de praticar (problem solving, preparação para entrevistas, etc).
2. **Nossas experiências:** um pouco da trajetória dos instrutores em competições de programação.
3. **Introdução ao C++:**
   * A estrutura básica do clássico `"Hello, World!"`.
   * Tipos de variáveis: `int`, `float`, `double`, `char`, `bool`.
   * Operadores aritméticos: `+`, `-`, `*`, `/`, `%`.
4. **Entrada e saída (stdin / stdout):**
   * Como ler dados com `cin >>`.
   * Como mostrar dados com `cout <<`.
