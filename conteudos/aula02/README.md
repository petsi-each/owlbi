# Aula 02: Condicional, loop e função

## 🦉 Sobre a aula
Neste segundo encontro da OwlBI, avançamos nos fundamentos da programação em C++. Começamos a dar inteligência aos nossos algoritmos utilizando estruturas de decisão, criamos repetições para lidar com múltiplos dados e aprendemos a organizar nossos códigos usando funções.

## 🎯 Tópicos abordados
* **Estruturas condicionais:** uso de `if`, `else if` e `else` para controle de fluxo.
* **Estruturas de repetição (loops):** implementação de laços `for`, `while` e `do..while`.
* **Funções:** como criar blocos de código reutilizáveis (`tipo nome_da_funcao()`) para evitar repetição e organizar o algoritmo.

## 💻 Problemas vistos em aula
Durante o encontro, desenvolvemos a lógica dos seguintes cenários:
* **Volume da TV:** uso de condicionais para classificar valores em "top demais", "ok" ou "slk num compensa" com base na terminação e paridade numérica.
* **Problema 2 (strings e condições):** impressão de "abacaxi", "beterraba" ou "-1" avaliando paridade e quantidade de dígitos.
* **Volume da TV++:** aplicação de loops para ler a entrada de uma quantidade determinada de vídeos sucessivamente.
* **Rodízio de pizza:** uso de loop com acumulador para somar o tamanho das fatias até estourar a capacidade máxima de um estômago.
